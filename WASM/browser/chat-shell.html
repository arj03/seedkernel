A signed-install bootstrap. The shell owns only the kernel, signature module, installer, WebRTC transport, and a sandboxed iframe. Apps arrive as signed WASM artifacts carrying their own UI; they can be loaded locally, exchanged peer-to-peer over the chat channel, and re-shared transitively — the author's signature on each install rides through every hop unchanged. See README §7 for the install model.
